# DIKWP DeepLearning LearnLab V1

Standalone, offline-first prototype for learning deep learning through evidence-based learning paths, interactive neural-network simulation, experiment tracking, project work, model debugging, AI tutor governance, and Learning Passport export.

## Quick start
Open `index.html` in a modern browser. No server, model API, database, account, or internet is required.

Optional CLI:
```bash
python run_demo.py examples/sample_learner.json --out outputs
```

## What it is
- a learning-path planner
- a deep-learning concept and prerequisite map
- a neural-network training simulator
- a PyTorch/Keras code skeleton workbench
- an experiment tracker
- a debugging playbook
- a project and paper reproduction planner
- an Evidence Ledger and Learning Passport generator

## What it is not
It is not an automated grader, employment evaluator, admissions system, research ethics approval, or production model validation system.
